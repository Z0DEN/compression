## Компиляция:

### (но проще скачать Huffman.exe из release)

```bash
cmake -B build -S . && cmake --build build
```

## Использование:

### При необходимости выдать права на запуск исполняемого файла

```bash
./Huffman.exe [-d для декодирования] [-f имя_файла - обязательный параметр]
```

#### например:

```bash
./Huffman.exe -f file
```

#### В результате появится файл `file.bin` и для его декодирования обратно:

```bash
./Huffman.exe -f file.bin -d
```

#### Результат - файл `file.bin.txt`
